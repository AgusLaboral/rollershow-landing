# Página de precios — esqueleto (sin números reales, pendiente de Marcelo)

**Por qué existe esta página**: la investigación de research (ver [RESEARCH.md](../RESEARCH.md) §2) encontró que casi nadie en el nicho publica precios, y el usuario que no encuentra un precio se va al competidor que sí lo muestra. La API pública de Rollershow **confirma que no devuelve precios** ("el vendedor cotiza por sus canales habituales, considerando promociones y ubicación") — así que estos números **no los puedo generar yo**, tienen que salir de Marcelo/ventas.

No armé el copy final para no inventar cifras. Dejo la estructura y qué necesito para completarla.

```
H1: Precios de cortinas roller a medida

[Párrafo de apertura: cómo se arma el precio — tela elegida + medida + sistema.
 Sin dar el número acá, dar el marco.]

[Tabla: familia (Roller / Verticales / Horizontales / Dúo / Tradicionales)

- Tela elegida (blackout liviana vs pesada, sunscreen, etc. — usar el
  gramaje/garantía como justificación de por qué varía, ya tengo esos
  datos reales de la API)

- Sistema (roller simple vs dúo vs verticales)
```

## Lo que necesito de Marcelo (ventas) para completar esto

1. Precio "desde" por familia de producto (Roller / Verticales / Horizontales / Dúo / Tradicionales), aunque sea un rango.
2. 3-4 ejemplos reales de cotización por medida típica (con y sin motorización).
3. Si el precio publicado incluye instalación y envío, o son aparte.
4. Si hay financiación/cuotas para mencionar.

Con esos 4 datos, esta página se termina de escribir en el momento.
